# MyfirstGame v0.0.7 - 首次发布

![游戏图标](icon.svg)

**发布日期**: 2025年11月8日  
**版本**: 0.0.7 (首个正式发布版本)  
**引擎**: Godot 4.5

## 🎮 游戏简介

《MyfirstGame》是一款2D射击生存游戏，玩家控制角色射击史莱姆敌人，通过击败敌人获取经验值并升级，每次升级可选择不同能力提升。

## ✨ 核心特性

- **快节奏射击**: WASD移动，静止时自动射击
- **动态难度**: 随时间推移，敌人生成速度加快，血量增加
- **升级系统**: 击败敌人获取经验，升级时三选一技能
- **爆炸效果**: 高伤害子弹产生范围伤害
- **屏幕震动**: 爆炸时的视觉反馈
- **完整音效**: 背景音乐、射击音效、敌人死亡音效

## 🎮 游戏操作

- **移动**: WASD键
- **射击**: 静止时自动射击
- **转向**: 左键向左转，右键向右转

### 技能选择
- **开火时间提升**: 射击间隔×0.75
- **子弹伤害提升**: 伤害×1.5
- **经验获取提升**: 经验值×2.0

## 📦 系统要求

- **操作系统**: Windows 10/11, macOS 10.14+, Linux
- **处理器**: 双核CPU
- **内存**: 2GB RAM
- **显卡**: 支持OpenGL 3.3
- **存储**: 100MB可用空间

## 🚀 安装与运行

1. 下载游戏压缩包并解压
2. Windows用户双击`MyfirstGame.exe`启动
3. 其他平台用户需使用Godot 4.5+打开`project.godot`文件
