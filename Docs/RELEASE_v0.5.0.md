# 像素弹雨 v0.5.0 - 重大版本更新

![游戏图标](../icon.svg)

**发布日期**：2025年12月15日  
**版本**：v0.5.0（重大功能更新）  
**引擎**：Godot 4.5

## 🎉 版本亮点

v0.5.0是《像素弹雨》迄今为止最大的版本更新，引入了完整的游戏Wiki文档系统，大幅改进了卡牌系统，并优化了整体游戏体验。

## 🌟 主要新增功能

### 📚 完整Wiki文档系统
- **结构化文档**：创建了完整的游戏Wiki体系
- **详细指南**：涵盖游戏机制、策略和开发文档
- **易于导航**：清晰的文档结构和交叉引用
- **持续更新**：支持版本迭代和内容扩展

### 🎴 卡牌系统重构
- **模块化架构**：基于SOLID原则的可扩展设计
- **标签分类系统**：支持卡牌分组和动态管理
- **工厂模式**：统一的卡牌创建和管理机制
- **信号通信**：松耦合的组件间通信

### 🎯 终极天赋系统完善
- **三种终极天赋**：追踪子弹、爆炸连锁、穿透
- **强化卡牌**：每个终极天赋都有对应的强化选项
- **平衡调整**：优化了数值和游戏平衡性
- **视觉反馈**：改进了特效和用户体验

### 👾 敌人系统优化
- **多种敌人类型**：普通史莱姆、巨型史莱姆、普通兔子、精英兔子
- **动态难度**：敌人血量和生成频率随等级增长
- **权重生成**：基于权重的随机敌人生成系统
- **行为模式**：统一的敌人AI和行为逻辑

## 🔧 技术改进

### 架构优化
- **信号总线系统**：实现松耦合的组件通信
- **敌人工厂模式**：统一的敌人生成和管理
- **配置管理系统**：音量和设置的持久化存储
- **键盘管理系统**：上下文感知的输入处理

### 性能优化
- **内存管理**：优化了对象创建和销毁
- **渲染优化**：改进了特效和动画性能
- **碰撞检测**：优化了碰撞检测算法
- **垃圾回收**：减少了内存碎片和GC压力

### 调试工具
- **游戏内控制台**：F1开启调试控制台
- **指令系统**：支持经验、等级、子弹类型等调试
- **实时测试**：无需重启即可测试功能
- **开发者友好**：提供了丰富的调试和测试工具

## 🎮 游戏机制改进

### 升级系统
- **初始选卡**：1级时从2张终极天赋中选择
- **常规升级**：2-9级从3张随机卡牌中选择
- **轮回选择**：10级时选择轮回选项
- **卡牌池管理**：根据选择动态调整可用卡牌

### 终极天赋详细说明

#### 追踪子弹
- **自动索敌**：子弹自动追踪最近的敌人
- **转向能力**：平滑的转向和追踪机制
- **强化选项**：转向速度、生存时间提升
- **适用场景**：新手友好，容错率高

#### 爆炸连锁
- **连锁爆炸**：击中敌人时触发范围爆炸
- **高爆发伤害**：可以一次性清除大量敌人
- **强化选项**：爆炸伤害、触发概率提升
- **适用场景**：高上限高回报的进阶选择

#### 穿透
- **子弹穿透**：子弹可以穿透多个敌人
- **直线清理**：适合处理直线排列的敌人
- **强化选项**：穿透伤害、触发概率提升
- **适用场景**：技术要求高但效果强大

### 轮回系统
- **轮回保留**：重置基础属性但保留终极效果
- **轮回重置**：完全重置所有属性，重新开始
- **策略选择**：需要根据游戏进度和目标选择
- **长期玩法**：支持多轮游戏和深度策略

## 📊 数值调整

### 基础数值
- **玩家移动速度**：70像素/秒（保持不变）
- **普通子弹速度**：300像素/秒（保持不变）
- **追踪子弹速度**：200像素/秒（保持不变）
- **升级公式**：`ceil(10 × (1.5^当前等级))`（保持不变）

### 敌人平衡
- **普通史莱姆**：血量2.0，经验3点（保持不变）
- **巨型史莱姆**：血量8.0，经验12点（保持不变）
- **普通兔子**：血量3.0，经验5点（保持不变）
- **精英兔子**：血量5.0，经验10点（保持不变）

### 卡牌数值
- **子弹伤害提升**：×1.5（保持不变）
- **开火速率提升**：×0.75（保持不变）
- **经验获取提升**：×2.0（保持不变）
- **强化卡牌**：优化了数值和效果

## 📋 已知问题

### 游戏机制
- 大量敌人同时出现时可能有轻微卡顿（已优化但未完全解决）
- 超高分辨率显示器上可能需要手动调整窗口大小
- 某些极端情况下敌人可能生成在屏幕外

### 调试工具
- 控制台历史记录在极长输入时可能出现异常
- 某些调试指令在特定状态下可能无效
- 性能监控功能在部分平台上可能不准确

### 平台特定
- Web版本不支持音频文件的自定义
- Linux版本在某些发行版上可能需要额外依赖
- macOS版本在某些硬件上可能出现性能下降
