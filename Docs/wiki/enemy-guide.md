# 敌人图鉴

## 👾 敌人系统概述

《像素弹雨》中的敌人系统采用工厂模式设计，支持多种敌人类型和动态难度调整。随着游戏进程推进，新的敌人类型会逐步解锁，为玩家带来不断变化的挑战。

## 🎯 敌人类型详解

### 1. 普通史莱姆 (Normal Slime)

**解锁等级**：游戏开始  
**敌人类型**：基础敌人  
**权重**：中等  

#### 基础属性
- **血量**：2.0（基础值）
- **移动速度**：50像素/秒
- **经验值**：3点
- **生成位置**：屏幕左右两侧随机
- **行为模式**：直线移动，朝向玩家

#### 行为特征
- **移动模式**：直线移动，持续朝向玩家位置
- **攻击方式**：接触玩家造成游戏结束
- **受击反应**：播放受击动画，短暂无敌时间0.25秒
- **死亡动画**：持续0.6秒的死亡特效

#### 策略建议
- **优先级**：低威胁，适合积累经验
- **处理方式**：保持距离，逐个击破
- **危险程度**：单只威胁低，数量多时需要注意

---

### 2. 巨型史莱姆 (Giant Slime)

**解锁等级**：5级  
**敌人类型**：精英敌人  
**权重**：中低  

#### 基础属性
- **血量**：8.0（基础值，普通史莱姆的4倍）
- **移动速度**：40像素/秒（比普通史莱姆慢）
- **经验值**：12点（普通史莱姆的4倍）
- **生成位置**：屏幕左右两侧随机
- **行为模式**：直线移动，朝向玩家

#### 行为特征
- **高血量**：需要多次攻击才能击败
- **缓慢移动**：移动速度较慢，但难以快速清除
- **高经验奖励**：击败后获得大量经验值
- **受击反应**：与普通史莱姆相同

#### 策略建议
- **优先级**：中等威胁，需要优先处理
- **处理方式**：集中火力快速击破
- **危险程度**：单只威胁中等，多只时形成防线

---

### 3. 普通兔子 (Normal Bunny)

**解锁等级**：5级  
**敌人类型**：快速敌人  
**权重**：中等  

#### 基础属性
- **血量**：3.0（基础值）
- **移动速度**：80像素/秒（比史莱姆快60%）
- **经验值**：5点
- **生成位置**：屏幕左右两侧随机
- **行为模式**：直线移动，朝向玩家

#### 行为特征
- **快速移动**：移动速度快，难以躲避
- **中等血量**：比普通史莱姆耐打
- **高机动性**：能够快速接近玩家
- **经验奖励**：击败后获得中等经验值

#### 策略建议
- **优先级**：高威胁，需要优先处理
- **处理方式**：使用追踪子弹或预判射击
- **危险程度**：单只威胁较高，群体时非常危险

---

### 4. 精英兔子 (Elite Bunny)

**解锁等级**：5级  
**敌人类型**：精英快速敌人  
**权重**：低  

#### 基础属性
- **血量**：5.0（基础值）
- **移动速度**：70像素/秒（比普通兔子稍慢）
- **经验值**：10点
- **生成位置**：屏幕左右两侧随机
- **特殊能力**：可以生成普通兔子

#### 行为特征
- **生成小怪**：定期生成普通兔子
- **高血量**：需要多次攻击才能击败
- **领导能力**：作为小队领袖出现
- **受击反应**：与普通敌人相同

#### 特殊机制
```gdscript
# 生成普通兔子的逻辑
func spawn_normal_bunny():
    if can_spawn and randf() < spawn_probability:
        var bunny = normal_bunny_scene.instantiate()
        bunny.position = position + Vector2(randf_range(-50, 50), 0)
        get_tree().current_scene.add_child(bunny)
```

#### 策略建议
- **优先级**：极高威胁，必须优先处理
- **处理方式**：集中火力快速击破，防止生成过多小怪
- **危险程度**：单只威胁高，生成小怪后威胁倍增

## 📊 敌人生成机制

### 权重系统

敌人生成采用权重随机系统：

| 敌人类型 | 权重 | 生成概率 |
|----------|------|----------|
| 普通史莱姆 | 10 | ~43% |
| 巨型史莱姆 | 3 | ~13% |
| 普通兔子 | 8 | ~35% |
| 精英兔子 | 2 | ~9% |

### 生成规则

1. **基础生成**：从所有启用的敌人中按权重随机选择
2. **位置随机**：屏幕左右两侧随机位置
3. **时间控制**：生成间隔随时间递减
4. **等级解锁**：5级解锁新敌人类型

### 生成间隔计算

```gdscript
# 生成间隔递减逻辑
Spawn_timer.wait_time -= 0.03 * delta
Spawn_timer.wait_time = clamp(Spawn_timer.wait_time, 0.2, 3.0)
```

- **初始间隔**：3.0秒
- **递减速度**：每秒减少0.03秒
- **最小间隔**：0.2秒
- **最大间隔**：3.0秒

## 🎯 敌人行为模式

### 移动模式

所有敌人都采用相同的移动模式：

```gdscript
# 基础移动逻辑
func _physics_process(delta):
    var direction = (player.position - position).normalized()
    position += direction * move_speed * delta
```

#### 移动特点
- **直线追踪**：始终朝向玩家当前位置
- **无障碍规避**：不会避开障碍物或其他敌人
- **持续追踪**：直到被击败或接触玩家

### 交互行为

#### 受击反应
- **无敌时间**：0.25秒
- **视觉反馈**：播放受击动画
- **击退效果**：无击退，保持位置

#### 死亡行为
- **死亡动画**：0.6秒的死亡特效
- **经验奖励**：立即给予玩家经验值
- **分数奖励**：增加游戏分数
- **尸体处理**：动画结束后移除

#### 生成行为（精英兔子）
- **生成条件**：随机时间间隔
- **生成位置**：精英兔子附近
- **生成数量**：一次生成一只普通兔子
- **生成限制**：有最大生成数量限制

## 🔧 敌人系统架构

### 工厂模式设计

```gdscript
class_name EnemyFactory
extends RefCounted

enum EnemyType {
    SlimeNormal = 1,
    SlimeGiant = 2,
    BunnyNormal = 3,
    BunnyElif = 4,
}

static func get_random_enemy_by_weight() -> PackedScene:
    # 权重随机生成逻辑
    var enabled_enemies = get_all_enabled_enemy()
    var total_weight = 0
    
    for enemy in enabled_enemies:
        total_weight += enemy.enemy_weight
    
    var random_value = randi_range(1, total_weight)
    var current_weight = 0
    
    for enemy in enabled_enemies:
        current_weight += enemy.enemy_weight
        if random_value <= current_weight:
            return get_scene_by_instance(enemy)
```

## 🎮 敌人应对策略

### 通用策略

1. **位置控制**：保持与敌人的适当距离
2. **优先级管理**：优先处理高威胁敌人
3. **资源利用**：合理使用终极技能和强化效果
4. **移动规划**：规划移动路线，避免被包围

### 分类型策略

#### 史莱姆类敌人
- **普通史莱姆**：适合积累经验，不必过于担心
- **巨型史莱姆**：需要集中火力，快速击破
- **共同弱点**：移动速度慢，容易躲避

#### 兔子类敌人
- **普通兔子**：需要快速反应，优先处理
- **精英兔子**：极高威胁，必须第一时间击破
- **共同弱点**：血量相对较低，可以快速击破

### 终极天赋应对

#### 追踪子弹
- **优势**：自动瞄准，无需精确操作
- **适用场景**：面对大量快速敌人时
- **配合策略**：专注于移动和躲避

#### 爆炸连锁
- **优势**：范围伤害，可以同时处理多个敌人
- **适用场景**：敌人密集聚集时
- **配合策略**：引诱敌人聚集后攻击

#### 穿透
- **优势**：直线穿透，适合处理队列敌人
- **适用场景**：敌人呈直线排列时
- **配合策略**：精确瞄准，最大化穿透效果
