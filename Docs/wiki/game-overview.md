# 游戏概述

## 🎮 游戏简介

《像素弹雨》(Pixel Bullet Rain)是一款使用Godot引擎开发的2D射击生存游戏。玩家需要控制角色射击不断出现的敌人，通过击败敌人获取经验值并升级，每次升级可以选择不同的能力提升。

## 🌟 核心特性

### 快节奏射击玩法
- 使用WASD键控制角色移动
- 角色静止时自动射击，移动时停止射击
- 支持普通子弹和追踪子弹两种弹药模式

### 动态难度系统
- 随着时间推移，敌人生成速度逐渐加快
- 敌人血量根据玩家等级动态增长
- 高等级解锁新的敌人类型

### 升级与技能选择
- 击败敌人获得经验值，积累足够经验后升级
- 每次升级可从三个随机技能中选择一个
- 包含基础强化、终极天赋和轮回选项

### 爆炸效果系统
- 高伤害子弹会产生爆炸效果，对周围敌人造成范围伤害
- 爆炸范围根据伤害值动态计算
- 配合屏幕震动效果增强视觉反馈

### 完整卡牌系统
- 模块化卡牌架构，支持多种强化类型
- 标签分类系统，便于管理和扩展
- 包含基础卡牌、终极天赋和轮回选项

## 🎯 游戏目标

玩家的主要目标是：
1. **生存挑战** - 尽可能长时间生存，躲避并击败不断出现的敌人
2. **等级提升** - 通过击败敌人获取经验值，提升等级解锁新内容
3. **能力强化** - 选择合适的卡牌组合，构建强大的战斗能力
4. **极限挑战** - 在高难度下挑战更高的分数和生存时间

## 🏆 游戏机制

### 基础操作
- **移动**：使用WASD键控制角色移动
- **射击**：角色静止时自动射击，移动时停止射击
- **转向**：按左键向左转，按右键向右转
- **暂停**：按ESC键暂停游戏

### 核心系统
1. **经验系统**：击败敌人获得经验值，积累足够经验后升级
2. **卡牌选择**：升级时从三张随机卡牌中选择一张
3. **难度递增**：随着游戏进行，敌人生成速度加快，血量增加
4. **终极天赋**：在特定等级选择终极技能，大幅提升战斗力
5. **轮回系统**：达到10级后可选择轮回，重置基础属性但保留终极效果

## 🎨 游戏风格

- **像素艺术风格**：采用复古像素艺术，营造经典街机游戏氛围
- **快节奏战斗**：紧张刺激的射击体验，考验玩家反应能力
- **策略选择**：丰富的卡牌组合，需要玩家根据局势做出明智选择
- **渐进难度**：平滑的难度曲线，让玩家逐步适应挑战

## 📊 游戏数据

### 基础数值
- **玩家移动速度**：70像素/秒
- **普通子弹速度**：300像素/秒
- **追踪子弹速度**：200像素/秒
- **升级公式**：`ceil(10 × (1.5^当前等级))`

### 敌人类型
- **普通史莱姆**：基础敌人，低血量低经验
- **巨型史莱姆**：高血量敌人，5级解锁
- **普通兔子**：快速移动敌人，5级解锁
- **精英兔子**：高血量快速敌人，5级解锁

## 🎮 游戏模式

目前游戏包含以下模式：
- **标准模式**：经典的生存挑战模式
- **调试模式**：开发者测试模式，包含调试控制台
