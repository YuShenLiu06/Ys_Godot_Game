# Spawn指令测试指南

## 指令格式
```
spawn <tscn> <x> <y>
```

## 测试用例

### 1. 生成普通史莱姆
```
spawn normal_slime 0 0
```
预期结果：在坐标(0, 0)位置生成一个普通史莱姆敌人

### 2. 生成测试敌人
```
spawn test_enemy 100 50
```
预期结果：在坐标(100, 50)位置生成一个测试敌人

### 3. 错误处理测试
```
spawn unknown_enemy 0 0
```
预期结果：显示错误信息，提示不支持的敌人类型

```
spawn normal_slime
```
预期结果：显示错误信息，提示缺少参数

## 功能验证点
1. 敌人是否在指定位置正确生成
2. 敌人是否具有正确的属性（血量、速度等）
3. 敌人是否能够正常移动和交互
4. 错误输入是否能够正确处理并显示友好的错误信息

## 使用方法
1. 在游戏中按F1打开调试控制台
2. 输入上述测试指令
3. 观察游戏中的敌人生成情况
4. 检查控制台输出的反馈信息
