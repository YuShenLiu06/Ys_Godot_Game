# 调试控制台使用说明

## 概述
调试控制台是一个用于游戏内调试和测试功能的工具，可以通过快捷键F1打开/关闭。

## 快捷键
- **F1**: 打开/关闭调试控制台
- **ESC**: 关闭调试控制台（当控制台打开时）
- **↑/↓**: 浏览指令历史记录

## 可用指令

### 1. 经验值管理
```
/exp add <amount>
```
- 功能: 增加指定数量的经验值
- 示例: `/exp add 100` - 增加100点经验值

### 2. 等级设置
```
/level set <amount>
```
- 功能: 设置角色等级
- 示例: `/level set 5` - 将等级设置为5

### 3. 子弹类型设置
```
/bullet model set <type>
```
- 功能: 设置子弹类型
- 参数:
  - 0: 普通子弹
  - 1: 追踪子弹
- 示例: `/bullet model set 1` - 切换到追踪子弹

### 4. 终极技能次数管理
```
/ultimate <type> add <amount>
```
- 功能: 增加指定终极技能的使用次数
- 参数:
  - type: 终极技能类型
    - tracking_bullet: 追踪子弹
    - explosion_chain: 爆炸连锁
    - penetrate: 穿透
  - amount: 增加的次数
- 示例: 
  - `/ultimate tracking_bullet add 2` - 增加追踪子弹终极技能2次
  - `/ultimate explosion_chain add 1` - 增加爆炸连锁终极技能1次
  - `/ultimate penetrate add 3` - 增加穿透终极技能3次

### 5. 控制台管理
```
/clear
```
- 功能: 清空控制台输出内容

```
/help
```
- 功能: 显示所有可用指令的帮助信息

## 使用注意事项
1. 控制台在游戏暂停时也可以使用
2. 指令不区分大小写
3. 使用↑/↓键可以浏览之前输入的指令历史
4. 控制台输出会自动滚动到最新内容
5. 输入错误或无效参数时会显示相应的错误提示

## 技术实现
- 控制台场景: `Scenes/UI/debug_console.tscn`
- 控制台脚本: `Scripts/manager/debug_console.gd`
- 集成到游戏主场景: `Scenes/manager/Game.tscn`
- 键盘管理器支持: `Scripts/manager/keyboard_manager.gd`

## 扩展性
控制台系统设计为可扩展的，可以通过以下方式添加新指令：
1. 在`_init_commands()`函数中注册新指令
2. 实现对应的指令处理函数
3. 在`_cmd_help()`函数中添加新指令的帮助信息
