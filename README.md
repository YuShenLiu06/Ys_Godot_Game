# MyfirstGame

一个使用Godot引擎开发的2D射击生存游戏，玩家需要控制角色射击不断出现的史莱姆敌人，通过击败敌人获取经验值并升级，每次升级可以选择不同的能力提升。

![游戏图标](icon.svg)

## 游戏特性

- **快节奏射击玩法**：使用WASD键移动，角色自动射击
- **动态难度系统**：随着时间推移，敌人生成速度逐渐加快
- **升级与技能选择**：通过击败敌人获取经验值，升级时从三个随机技能中选择一个
- **爆炸效果**：高伤害子弹会产生爆炸效果，对周围敌人造成范围伤害
- **屏幕震动**：爆炸时产生视觉反馈，增强游戏体验
- **音效系统**：包含背景音乐、射击音效、敌人死亡音效等

## 游戏玩法

### 基础操作

- **移动**：使用WASD键控制角色移动
- **射击**：角色静止时自动射击，移动时停止射击
- **转向**：按左键向左转，按右键向右转

### 游戏机制

1. **生存挑战**：躲避并击败不断出现的史莱姆敌人
2. **经验系统**：击败敌人获得经验值，积累足够经验后升级
3. **技能选择**：每次升级可从三个随机技能中选择一个：
   - 开火时间提升：减少射击间隔
   - 子弹伤害提升：增加每次射击的伤害
   - 经验获取提升：增加获得的经验值
4. **难度递增**：随着游戏进行，敌人生成速度逐渐加快，血量也会增加

### 核心系统

1. **游戏管理器** ([`Game_manager.gd`](Scripts/manager/Game_manager.gd))
   - 控制敌人生成和难度递增
   - 管理分数、经验和等级系统
   - 处理升级时的技能选择界面
   - 实现屏幕震动效果的全局接口

2. **玩家系统** ([`player.gd`](Scripts/manager/player.gd))
   - 处理玩家移动和动画
   - 控制射击逻辑
   - 管理游戏结束状态

3. **敌人系统** ([`enemy.gd`](Scripts/enemy/enemy.gd))
   - 实现敌人基础行为和属性
   - 处理受击和死亡逻辑
   - 支持不同类型的敌人扩展

4. **子弹系统** ([`bullet.gd`](Scripts/bullet/bullet.gd))
   - 实现子弹移动和碰撞检测
   - 支持爆炸效果和范围伤害

5. **技能选择系统** ([`selection.gd`](Scripts/selections/selection.gd))
   - 实现升级时的技能选择界面
   - 管理不同技能的效果

### 设计模式

- **信号系统**：使用Godot的信号系统实现松耦合的组件通信
- **继承与多态**：通过敌人基类实现不同类型敌人的统一管理
- **单例模式**：游戏管理器作为全局状态管理器
- **工厂模式**：用于动态创建敌人和子弹实例

## 开发环境

- **引擎版本**：Godot 4.5
- **渲染器**：Forward Plus
- **编程语言**：GDScript

## 安装与运行

1. 确保已安装Godot 4.5或更高版本
2. 克隆或下载本项目
3. 使用Godot编辑器打开`project.godot`文件
4. 点击运行按钮启动游戏
